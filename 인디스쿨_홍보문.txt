📣 [무료/설치없음] 교실 자리바꾸기 앱 공유합니다!

안녕하세요, 선생님들!
자리 바꿀 때마다 제비뽑기 준비하시느라 번거로우셨죠?
클릭 한 번으로 자리 배치가 끝나는 웹 앱을 만들어 공유드립니다. 😊


🪑 자리바꾸기란?

설치 없이 브라우저에서 바로 쓸 수 있는 교실 자리 배치 도구입니다.
학생 명단만 입력하면 랜덤 배치가 바로 완성됩니다!


✨ 주요 기능

📌 4가지 배치 유형
  - 시험대형, 짝대형, U대형, 자유배치(드래그로 책상 직접 배치)

📌 고정 자리
  - 특정 학생을 원하는 자리에 고정할 수 있습니다.

📌 분리 규칙
  - "이 두 학생은 최소 ○칸 떨어뜨려 주세요" 설정이 가능합니다.
  - 거리는 가로+세로 칸 수 합으로 계산되어, 대각선도 2칸으로 정확하게 반영됩니다.

📌 성별 기반 배치 ✨NEW
  - 학생별 성별을 지정하고, 동성끼리 또는 이성끼리 앉기 규칙을 적용할 수 있습니다.
  - 짝대형에서는 짝꿍끼리, 그 외 대형에서는 상하좌우 인접 자리에 적용됩니다.

📌 이전 자리 재배치 방지 ✨NEW
  - 배치 기록이 자동 저장되어(최대 5회), 다음 뽑기 시 같은 자리에 같은 학생이 앉지 않습니다.
  - 최근 1~3회 범위 설정, 기능 끄기/기록 초기화 모두 가능합니다.

📌 결과 저장/공유
  - JSON 파일, 이미지(PNG), 인쇄 모두 지원합니다.
  - 설정 내보내기/가져오기로 학기 초 세팅을 간편하게 옮길 수 있습니다.


💻 사용 방법

1. 아래 링크에 접속합니다. (PC/태블릿 권장)
2. 교사 설정에서 학생 명단, 배치 유형, 규칙을 설정합니다.
3. "학생 화면으로" 이동 후 "자리 뽑기!" 버튼을 누르면 끝!
4. 전체 화면으로 띄워 놓으면 TV/프로젝터로 바로 보여줄 수 있습니다.

💡 참고사항

- 회원가입, 로그인, 설치 모두 필요 없습니다.
- 데이터는 브라우저에 저장되어 개인정보가 외부로 전송되지 않습니다.
- 인터넷 없이도 한 번 접속 후 오프라인 사용이 가능합니다.
- 기존에 저장된 데이터가 있어도 업데이트 후 그대로 사용 가능합니다.
